Algorithm:
Step 1: Input Handling
 Get the input from the user.
Step 2: Tokenization
 Split the input line into words using space characters as delimiters.
 Store the words in a list or array for further processing.
Step 3: Filtering Words starting with “b”
 Initialize an empty list to store words starting with "b".
 Iterate through the list of words.
 Check if the current word starts with the letter "b".
 If it does, add the word to the list of words starting with "b".
Step 4: Filtering Words Ending with "ED"
 Initialize an empty list to store words ending with "ED".
 Iterate through the list of words.
 Check if the current word ends with the letters "ED".
 If it does, add the word to the list of words ending with "ED".
Step 5: Print the Results
 Display the list of words starting with "b".
